## 第一章：录入您的医院单据（批量操作指南）

### 方法：OCR 拍照识别（推荐）

**适用场景**：手头有几十张纸质单据，想快速数字化

**操作步骤**：

1. **注册/登录**
   - 打开 `http://69.5.21.128:8050`
   - 使用手机号注册（首次需要）
   - 登录后进入首页

2. **进入「病历录入」**
   - 点击底部导航栏的「添加」或「病历」按钮
   - 选择 **「OCR 拍照识别」**

3. **拍照上传**
   - 对准医院单据（门诊病历、检查报告、处方单、住院小结等）
   - 确保光线充足、文字清晰、四边对齐
   - 点击拍照 → 系统自动识别文字

4. **确认识别结果**
   - 系统会提取：就诊日期、医院名称、科室、诊断结果、用药信息
   - **⚠️ 重要**：请人工核对识别结果，特别是：
     - 日期是否正确（OCR 可能把 "2024" 识别为 "2021"）
     - 药品名称和剂量（关乎用药安全）
     - 医生签名区的文字（通常识别率低）
   - 修改错误字段 → 点击「保存」

5. **批量录入技巧**
   - 建议按 **时间顺序** 从近到远录入
   - 每次拍照后先点「保存并继续」，可连续录入多张
   - 同类单据（如化验单）可以一起拍，但需分开保存（每张一个记录）

### 方法：手动录入（OCR 识别失败时备用）

**适用场景**：手写病历、低质量复印件、OCR 识别率低于 80%

**操作步骤**：
1. 进入「病历录入」→ 选择「手动录入」
2. 填写必填项：患者姓名、就诊日期、医院、科室
3. 填写诊断信息（可输入多个诊断，用逗号分隔）
4. 如知道 ICD-10 编码可手动输入，否则留空（系统会尝试自动匹配）
5. 添加用药信息（药品名、剂量、频次、疗程）
6. 上传单据照片作为附件（即使 OCR 失败，照片也能留存备查）
7. 点击「保存」

### 录入完成后的整理

录入完成后，系统会自动：
- 📅 按时间生成**就诊时间线**
- 🏷️ 为每条记录标注 **ICD-10 疾病编码**
- 📊 统计各科室就诊次数、疾病分布

您可以在「时间线」页面查看所有记录的时间顺序视图。

## 第二章：让父亲查看自己的病历（家人共享功能）

### 场景说明

您作为子女，想帮父亲管理他的病历，同时让他自己也能随时查看。系统支持 **跨用户家人共享**，即父亲需要注册自己的账号，您向他发送邀请，他接受后双方即可互相查看病历。

### 操作步骤

#### 第一步：为父亲注册独立账号

1. 让父亲在他的手机上访问 `http://69.5.21.128:8050`
2. 点击「注册」，用他的手机号注册（例如：父亲的手机号 138****8888）
3. 设置密码，完成注册
4. **⚠️ 重要**：父亲的账号也需要录入他自己的病历（见第一章）

#### 第二步：发送家人共享邀请

1. 在您的手机上登录您的账号
2. 进入 **「我的」→「家人共享」**
3. 点击「邀请家人」
4. 输入父亲的手机号（138****8888）
5. 选择关系「父亲」
6. 点击「发送邀请」

#### 第三步：父亲接受邀请

1. 父亲在他的手机上登录他的账号
2. 进入 **「我的」→「家人共享」→「收到的邀请」**
3. 看到您的邀请后，点击「接受」
4. 双方绑定成功

#### 第四步：查看家人病历

绑定成功后：
- 您在「家人共享」页面点击父亲的头像，即可查看他的所有病历
- 父亲也可以在他的「家人共享」页面查看您的病历（双向可见）
- 所有病历数据保持同步更新

### 隐私与安全

- 只有接受邀请后双方才可见
- 任何一方可以在「家人共享」中随时「解除关系」
- 管理员后台会记录所有共享操作（用于安全审计）

## 第三章：系统独特功能介绍

### 1. 🤖 AI 报告解读

**位置**：在任意检查报告详情页点击「AI 解读」

**功能**：上传 CT、B超、血常规等检查报告，AI 会自动：
- 提取关键指标（如白细胞计数、血糖值等）
- 标注异常项（↑ 高于正常值、↓ 低于正常值）
- 给出通俗易懂的解读说明（非诊断建议，仅供参考）
- 提示需要关注的复查项目

**⚠️ 注意**：AI 解读仅供参考，不能替代医生诊断。如有严重异常，请务必就医。

### 2. 📋 就诊准备清单

**位置**：「AI 助手」→「就诊准备」

**功能**：输入就诊科室和症状，AI 自动生成：
- 需要携带的资料清单（身份证、医保卡、既往检查报告等）
- 建议提前准备的问题（方便高效问诊）
- 就诊前注意事项（如空腹要求等）

### 3. 📄 病例摘要

**位置**：「AI 助手」→「病例摘要」

**功能**：选择一段时间范围内的病历，AI 自动生成：
- 就诊概况总结
- 用药清单汇总（去重，显示当前在用药）
- 关键诊断时间线
- 方便转诊时给新医生快速了解病史

### 4. 💊 用药提醒

**位置**：「健康」→「用药提醒」

**功能**：
- 添加每日用药计划（药品名、剂量、时间）
- 到点推送提醒（需保持页面打开或添加至手机桌面）
- 记录服药打卡

**⚠️ 注意**：当前为网页版，提醒依赖浏览器通知，建议同时设置手机闹钟作为备份。

### 5. 🩺 体检异常管理

**位置**：「健康」→「体检异常」

**功能**：
- 记录历年体检中的异常指标
- 追踪异常指标变化趋势（轻度/中度/重度分级）
- 设置复查提醒

### 6. 💉 疫苗管理

**位置**：「健康」→「疫苗接种」

**功能**：
- 记录已接种疫苗（新冠、流感、HPV 等）
- 根据出生日期自动生成国家免疫规划接种计划
- 设置下一针提醒
- 查看全家疫苗接种情况（结合家人共享）

### 7. 📊 管理后台（仅管理员）

作为管理员，您可以访问：
- **运营看板**：总用户数、日活跃用户、功能使用率统计
- **用户管理**：查看所有注册用户、搜索、状态管理
- **用户详情**：查看单个用户的注册时间、活跃情况、病历数量

## 第四章：常见问题与排错

### Q1：OCR 识别结果不准确怎么办？

**原因**：
- 拍照时手抖/光线暗/反光
- 手写体或字迹潦草
- 表格线干扰识别

**解决方法**：
1. 重新拍照：确保光线均匀、单据平整、文字清晰
2. 手动修正：在识别结果页面直接编辑错误字段
3. 换用「手动录入」：对于手写病历，直接手动输入更可靠

### Q2：录入后找不到某条记录了？

**可能原因**：
- 录入时患者姓名选错了（比如选了父亲的名字）
- 就诊日期填写错误，导致在时间线中位置不对

**解决方法**：
1. 进入「时间线」→ 使用顶部搜索框搜索关键词（医院名、诊断名）
2. 检查「筛选」条件是否设置了日期范围

### Q3：父亲收不到家人共享邀请？

**排查步骤**：
1. 确认父亲已注册账号（未注册的手机号无法收到邀请）
2. 确认输入的手机号无误
3. 让父亲检查「我的」→「家人共享」→「收到的邀请」
4. 如果仍看不到，取消后重新发送

### Q4：AI 功能无响应或报错？

**原因**：AI 服务依赖火山方舟 API，可能因网络波动或配置问题暂时不可用

**解决方法**：
1. 刷新页面后重试
2. 如持续报错，请检查服务器日志或联系管理员
3. 当前 AI 配置状态可在后端日志中查看：`Ark AI: NOT CONFIGURED` 表示未配置（当前系统可能有此问题）

### Q5：数据安全吗？会丢失吗？

**数据存储**：
- 所有数据存储在服务器 SQLite 数据库中
- 服务器位于火山引擎云，有定期备份
- 建议定期导出个人数据（「我的」→「数据导出」）作为本地备份

**导出格式**：
- JSON：完整数据结构，适合技术备份
- CSV：Excel 可打开，适合查看和打印

### Q6：注册时间显示 1970/1/21？

**原因**：系统内部时间戳为 0 时的显示（Unix 时间起点）

**解决方法**：不影响使用。如需修正，管理员可在数据库中更新 `created_at` 字段为当前时间戳。

### Q7：管理员后台登录不了？

**排查**：
1. 确认访问地址是 `http://69.5.21.128:8050/admin/login`
2. 确认使用管理员手机号 `18600963966`
3. 确认密码是 `Admin123456`（注意末尾是 6，不是 5）
4. 如提示 Session Expired，这是页面加载时的正常检查，直接输入账号密码登录即可

## 📌 使用建议

1. **定期备份**：每月导出一次 CSV 备份到本地电脑
2. **录入规范**：每次就诊后立即录入，避免堆积
3. **照片留存**：即使 OCR 成功，也保留原始单据照片作为附件
4. **敏感信息**：不要在备注中记录银行卡号、密码等无关信息
5. **多人使用时**：建议每个家庭成员各自注册独立账号，再通过家人共享绑定
